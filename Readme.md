### 介绍
1. 本项目意在制作一个供自己使用的简易2D游戏引擎
2. 借助netcore原生的高性能，制作工作细胞

### 系统规划

- 引擎
  - 框架
    - FosterFramework
    - Arch
  - 功能
    - [x] 基于Transform的坐标系统
    - [x] 渲染系统
    - [x] 正交相机系统 
    - [x] 简单资源管理系统
      - [x] 图集
    - [ ]简单的空间分区
    - 动画系统
      - 状态机
      - 帧动画
    - UI系统
    - 粒子系统
    - 音频系统
    - 保存系统
- 内容
  - 基于ecs搭建游戏
- Runner:发布的运行程序
- Editor:游戏编辑器
  - 编辑器
    - Inspector
    - 显示游戏运行的窗口
