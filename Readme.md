### 介绍
1. 本项目意在制作一个供自己使用的简易2D游戏引擎
2. 借助netcore原生的高性能，制作工作细胞

### 系统规划

- 引擎
  - 框架
    - FosterFramework
    - Arch
  - 功能
    - 基于Transform的场景图
    - UI系统
    - 渲染系统
    - 粒子系统
    - 空间分区
    - 资源管理系统
    - 保存系统
    - 音频系统
- 内容
  - 基于ecs搭建游戏
- Runner:发布的运行程序
- Editor:游戏编辑器
  - 利用ImGui制作一个简单的编辑器
  - 显示游戏运行的窗口
